DIAGNE Salla

Instruction :
	- se placer à la racine du projet (diagne_tp4/oie) et exécuter la commande :
		java -jar mygoosegame.jar p1 p2 ... pn
	Exemples :
		java -jar mygoosegame.jar A pour lancer un jeu de l'oie avec un joueur nommé A
		java -jar mygoosegame.jar A B pour lancer un jeu de l'oie avec deux joueurs nommés A et B
		java -jar mygoosegame.jar A B C pour lancer un jeu de l'oie avec trois joueurs nommés A, B et C
		etc...
