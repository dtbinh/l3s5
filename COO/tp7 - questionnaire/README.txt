Instructions :
	1) pour exécuter le questionnaire simple : se placer à la racine du projet (dossier diagne_tp7/questionnaire) et taper la commande
	   java -jar myquestionnaire.jar fileName (avec fileName le nom du fichier dont on extrait les informations servant à initialiser le questionnaire)

	2) pour exécuter le questionnaire avec IHM : se placer à la racine du projet (dossier diagne_tp7/questionnaire) et taper la commande
	   java -jar myquestionnaire-ihm.jar fileName (avec fileName le nom du fichier dont on extrait les informations servant à initialiser le questionnaire et créer l'interface graphique)
